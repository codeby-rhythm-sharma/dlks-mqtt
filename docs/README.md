# Documentation

The `docs` directory contains technical documentation, architectural references, implementation details, simulation reports, performance evaluations, and research material associated with the DLKS-MQTT framework.

This includes:
- Lightweight MQTT security workflow analysis
- Ephemeral session key generation methodology
- Replay attack and MitM mitigation approach
- Performance benchmarking and comparative evaluation
- IoT communication security architecture
- Experimental simulation findings and future enhancements

The documentation serves as a supporting reference for the practical implementation and security design decisions used throughout the project.
