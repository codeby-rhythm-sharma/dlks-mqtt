# Simulation Module

This module contains simulation workflows and IoT communication testing environments used for evaluating the DLKS-MQTT framework.

The simulation layer is designed to analyze communication security, performance efficiency, and attack resilience in constrained IoT environments.

## Simulation Objectives

- Secure MQTT communication testing
- IoT device interaction modeling
- Replay attack simulation
- Lightweight communication benchmarking
- Communication overhead evaluation
- Session validation testing
- Performance analysis under constrained environments

## Testing Environment

The framework is designed for lightweight IoT simulation environments using tools such as:

- Contiki OS
- Cooja Simulator
- MQTT communication workflows
- Lightweight security testing scenarios
