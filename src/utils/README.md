# Utility Module

This module contains helper utilities and reusable supporting components used across the DLKS-MQTT framework.

The utilities layer is designed to improve modularity, maintainability, and consistency throughout the framework implementation.

## Included Utilities

- Configuration handling
- Communication helpers
- Logging utilities
- Common reusable functions
- MQTT connection helpers
- Data formatting support
- Session management utilities

## Purpose

The utility module supports lightweight and organized development practices for secure MQTT-based IoT communication environments.
