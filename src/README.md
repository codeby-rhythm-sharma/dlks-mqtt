# Source Code

This directory contains the core implementation components of the DLKS-MQTT framework.

## Planned Modules

- `client/` — MQTT client-side communication logic
- `broker/` — Secure broker interaction workflow
- `security/` — Session key generation, hashing, and authentication mechanisms
- `simulation/` — IoT simulation and testing components
- `utils/` — Helper utilities and configuration modules

The implementation focuses on lightweight secure communication for resource-constrained IoT environments.
